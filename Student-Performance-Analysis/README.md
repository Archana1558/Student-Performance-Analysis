Student Performance Analysis

This project analyzes student performance using Python and SQL to identify key factors affecting academic outcomes.

Tools Used
- Python (Pandas, Matplotlib, Seaborn)
- SQL (SQLite)
- Jupyter Notebook

Analysis Performed
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- SQL-based insights

Key Insights
- Study hours significantly impact performance
- Attendance is a strong indicator of success
- Study methods influence overall scores
- Internet access has a moderate effect on performance
